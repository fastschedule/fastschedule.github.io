FAST Timetable

About

This is an independent actively maintained web tool that currently works only on the Bachelor's Computing Department of FAST Islamabad. The goal is to not look at the annoying excel doc again and again which is the official way of checking your timetable

I originally made this tool for myself because the excel sheet never loads in my phone (idk why) but then decided to make it public.
The data updates automatically every 15 minutes and before checking make sure to refresh the page as the previous one might be cached by the browser.

If you find a bug or some inaccurate info or suggestions feel free to contact me (details given below in the footer)

Features Planned (in no particular order):

  • Add support for MS
  • Add support for Repeat Courses
  • Add support for Elective Courses
  • Notification system for holidays
  • Make a similar thing for the Management and Electrical students

How reliable is it??

I would say 99.9% of the time the info is accurate and the last 0.1% can be due to sudden unpredictable changes FAST decides to make in their format, but these kind of issues are usually resolved within an hour and users are notified immediately; I have tried my best to account for all edge cases but there is still a very small chance that it might report something wrong so please double check with the official FAST NUCES sources before making any important decisions based on this tool and contact me if you find any issues.
